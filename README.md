## 流程自动化原理

将可视化界面的数据生成一个一个 js 代码，然后用 nodejs 运行这个项目js,从而实现自动化流程的运行管理
指令 ———— 也就是提供每个js 可运行的函数，这个函数可以运行nodejs 的任何api
